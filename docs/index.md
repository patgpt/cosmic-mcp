## 🚀 Quick Start

```bash
# Install and run with npx (recommended)
npx -y cosmic-mcp

# Or install globally
npm install -g cosmic-mcp
cosmic-mcp
```

Just add your Cosmic credentials and you're ready to manage content with AI!
